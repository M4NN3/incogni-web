# incogni-web
Simple web app to show movies info using Spring Framework + Thymeleaf

## Stack:
* Spring Framework
* Srping Security
* Firebase/Firestore
* Thymeleaf
* TMDB API (only for movie cast crew)
* BulmaCSS
